## quickjs


quickjs is a js engine written by Fabrice Bellard.

### get the code, running, testing, logging, lsp

```bash
git clone https://github.com/bellard/quickjs.git

sudo apt install bear make clangd

whereis clangd
{
    "clangd.path": "/usr/bin/clangd"
}

bear -- make -j4



```
