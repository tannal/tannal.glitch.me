# AMD

这些新闻标题主要涉及 AMD (Advanced Micro Devices) 的收购活动。我将简要介绍一下这些被收购的公司：

Pensando：一家专注于数据处理单元（DPU）的公司。这次收购帮助 AMD 进入了 DPU 市场。
